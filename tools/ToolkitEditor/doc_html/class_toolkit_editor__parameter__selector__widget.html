Detailed Description

Table-based widget to show a list of parameters and select one of them.

The implementation of this widget relies on the table view + table model concept.

See classes ToolkitEditor_parameter_table_view (the table view) and ToolkitEditor_parameter_model (the model), on which this class relies.
