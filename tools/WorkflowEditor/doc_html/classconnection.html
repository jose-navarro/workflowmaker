◆ connection()

connection::connection ( wfme_subnode_data & startNode,
wfme_subnode_data & endNode,
QGraphicsItem * parent = nullptr )

Constructor.

Parameters
[in]startNodeThe information describing the starting endpoint of the connection.
[in]endNodeThe information describing the final endpoint of the connection.
[in]parentThe parent of this item.

Note that both parameters, startNode and endNode contain information about:

  • The DiagramItem items to which these endpoints correspond.
  • The type of connector (input / output).
  • The slot number (for instance, the first, third, ..., slot).

Compute the scene coordinates of both the initial and final enpoints, that is, the two DiagramItems that the used decided to connect drawing a line.

Parameters
[out]startScene coordinates of the initial endpoints.
[out]endScene coordinates of the final endpoint.

Note: these coordinates belong to the center of the specific slots of the task / repository where the connection starts and ends.

Retrieve the subnode data related to the final endpoint of the connection.

Returns
A copy of the subnode data related to the final endpoint of the connection.

The subnode data contains information about the actual DiagramItem involved in the connection, whether this is an input or output one, and the specific slot of the task / repository where the connection starts or ends.
